# Codevocado UI/UX Design Projects

A collection of UI/UX design projects created during my internship at Codevocado using Figma.

## 🎨 Featured Design Projects

### 1. AI-Powered Face Attendance System

Designed UI screens for an AI-powered attendance system with facial recognition and biometric authentication features.

### 2. Learning Management System (LMS)

Designed interfaces for an interactive learning management system with course management, live and recorded classes, assignments, examinations, attendance, and student/teacher dashboards.

### 3. Online Examination Portal

Designed a secure online examination platform with AI-powered facial recognition, online proctoring, exam management, question banks, result analysis, and student performance tracking.

### 4. HRMS & ERP System

Designed interfaces for an integrated Human Resource Management and Enterprise Resource Planning system, including employee management, attendance, leave management, payroll, reports, and administrative operations.

### 5. School Management System

Designed an all-in-one school management platform with admission and attendance automation, learning management, fee management, timetable scheduling, payroll, reports, and role-based dashboards.
